Primeiro Teste em JS

1-> Crie uma página HTML básica com título, descrição da atividade e espaço para inserção de dados pelo usuário.
2-> Adicione dois campos de entrada (input): um para o preço original do produto e outro para a porcentagem de desconto.
3-> Inclua um botão que, ao ser clicado, execute uma função javascript para calcular o preço final após o desconto.
4-> A função deve usar operadores para calcular o valor do desconto e o preço final.
5-> Use estruturas condicionais if-else para verificar se os valores inseridos são válidos (por exemplo, preços ou porcentagens negativas não são permitidas).
6-> Exiba o resultado usando console.log e alert para mostrar o preço original, o valor do desconto e o preço final.


Prática:


Calculadora de desconto
